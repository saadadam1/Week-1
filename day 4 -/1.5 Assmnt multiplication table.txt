5. Write a program to accept a number and print table for it.
      input : 2
                  2*1=2
                  2*2=4
           
                  2*10=20


begin 
  
  number num, i=1 , sum
 print("enter a number")
 input num

 while i<=10
 {
   sum=num*i
   print(num,"*",i,"=",sum)

 }
